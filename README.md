# Web Technologies Project

This project is a Sports Facility Booking System developed as part of the Web Technologies course.

## Architecture
The project follows the MVC (Model–View–Controller) architecture.

## Team Workflow
We follow a Gitflow-based workflow:
- feature branches are created from `dev`
- feature branches are merged into `dev`
- `dev` is promoted to `stage`
- `stage` is merged into `main`

## Repository Structure
- mvc_project/controllers
- mvc_project/models
- mvc_project/views
- mvc_project/uploads
